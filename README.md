# Marketing Funnel Analysis

## Objective
Analyze user behavior across the marketing funnel to identify drop-off points and improve conversion performance.

## Dataset
https://www.kaggle.com/datasets/alperenmyung/social-media-advertisement-performance

## Key Features
- Event-level data analysis (impression, click, purchase)
- Funnel KPI calculation (CTR, conversion rate)
- Data processing workflow (clean → transform → aggregate)
- Visualization of user behavior

## Key Insights
- Significant drop-off from impressions to clicks indicates low engagement
- Purchase rate is very low, suggesting issues in final conversion stage
- Engagement events (likes, shares, comments) indicate user interest beyond clicks

## Tools
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## Project Structure
- Data loading and cleaning
- KPI calculation
- Funnel visualization
- Business insights and recommendations
